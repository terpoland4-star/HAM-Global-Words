🌐 Comprendre le Web

Le navigateur agit comme un interprète. Il reçoit du HTML et le transforme en interface visuelle.

HTML = HyperText Markup Language

Ce n'est pas un langage de programmation, mais un langage de structuration.

📄 Structure d'une page HTML

<!DOCTYPE html>
<html>
  <head></head>
  <body></body>
</html>
  • DOCTYPE : définit HTML5
  • html : contient tout
  • head : infos invisibles
  • body : contenu visible

🏗️ Les balises HTML

<balise>contenu</balise>
  • <h1> : titre principal
  • <p> : paragraphe
  • <a> : lien
  • <img> : image
  • <div> : structure
<a href="https://google.com">Google</a>
<img src="image.jpg" alt="description">

⚙️ Les attributs

<balise attribut="valeur">
  • href → lien
  • src → source image
  • alt → description image
  • class → style CSS
  • id → identifiant unique

🔣 Les symboles HTML

  • < et > → balises
  • / → fermeture
  • = → assignation
  • " " → valeur
<a href="site.com">

❌ Erreurs courantes

<h1>Test
<p>Bonjour

Correction :

<h1>Test</h1>
<p>Bonjour</p>
